# Website of John Dengis

## Context

- We are creating a personal website for John Dengis.
- John Dengis is a software engineer by trade and spent the better part of 7 years as an engineering leader at Instacart.
- John has also made YouTube videos previously at the following channel https://www.youtube.com/@JohnDengisYT. These were primarily cooking, culture and travel videos.
- John also likes making music was in a few bands in the past, including https://www.youtube.com/@forbiddenconjecture, a progressive metal band. John still likes to play guitar and tinker with gear and signal chains.


## Task

- We will create a unique and personalized website for John Dengis
- The expected domain for this website will be https://john.deng.is/

## Requirements

- Reusable component elements of the design should be but in ./app/components/ui/ folder. This is things like reusable buttons, cards, fields, toggles etc.
- Website should support both dark mode and light mode.
- The website should have an index page with general overview, a more detailed "about" page, a list of projects referring to specific projects that John has made, such as his GitHub repos, YouTube etc., a blog with RSS feed, and small simplified resume (look at this for some inspiration https://tailwindcss.com/plus/templates/spotlight/preview). There should also be social links with the appropriate svg icons.
    - X: https://x.com/jadengis/
    - GitHub: https://github.com/jadengis/
    - LinkedIn: https://www.linkedin.com/in/john-dengis/
    - YouTube: https://www.youtube.com/@JohnDengisYT/
- For color inspiration, John likes black and red a lot with accents of white. Bright chromatic gradients are nice and a splash of color always makes things look good. 
- Blog posts should be written in markdown and should contain jsonld content for better SEO indexing characteristics.
- Generated sitemap.xml
- Simple site map in the footer.
- Website should have good SEO and GEO practices built in.
